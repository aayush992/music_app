# Android Music Player App

An Android application that streams and plays music using the YouTube API.

## Features
- Search and browse songs dynamically
- Play music using YouTube API
- RecyclerView-based UI for smooth scrolling
- Dedicated player screen with playback controls

## Architecture
- Used Retrofit for API integration
- Structured data flow using model classes
- RecyclerView + Adapter pattern for UI rendering

## Tech Stack
- Kotlin
- Android SDK
- Retrofit
- YouTube API
- RecyclerView

## Challenges
- Handling asynchronous API calls without blocking UI
- Managing smooth playback and navigation between screens
- Ensuring responsive UI during network requests

## Learning
This project helped me understand API integration, async handling, and building responsive Android UI.
